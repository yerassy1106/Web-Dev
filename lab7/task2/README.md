# Vehicle Management System

A Python-based demonstration of **Object-Oriented Programming (OOP)** principles. This project showcases how to use inheritance and polymorphism to manage a fleet of different vehicle types.

## 📌 Project Structure

The project consists of two main Python files:
* **`model.py`**: Contains the class definitions (`Vehicle`, `Car`, and `Truck`).
* **`main.py`**: The entry point of the application that initializes the objects and executes the logic.

## 🚀 How to Run

### 1. Prerequisites
Make sure you have **Python 3.x** installed on your machine.

### 2. Setup
Place both `main.py` and `model.py` in the same folder. 

**Note:** Ensure your `main.py` starts with the following import:
```python
from model import Car, Truck
